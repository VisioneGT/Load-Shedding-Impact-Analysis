# ⚡ Load Shedding Impact Analysis

A data analytics project measuring how Eskom load shedding affected the daily
revenue of a Johannesburg retail shop across January, February, and March 2024.
Built to demonstrate data analysis, statistical modelling, and business insight
skills for a junior data analyst portfolio.

---

## What This Project Does

- Measures the revenue impact of each load shedding stage (1–6)
- Calculates the Pearson correlation between load shedding hours and revenue
- Uses linear regression to predict daily revenue from load shedding data
- Detects anomalous revenue days using a 7-day rolling average method
- Forecasts April 2024 revenue under four different load shedding scenarios
- Estimates total rand value lost to load shedding over 3 months
- Presents everything in an interactive multi-page Dash dashboard

## How to Run

### Step 1 — Install dependencies
```bash
pip install pandas plotly dash scikit-learn jupyter openpyxl
```

### Step 2 — Run the notebooks (recommended first)
```bash
cd notebooks
jupyter notebook
```
Run notebooks 01 through 04 in order. Notebook 01 generates `data/merged.csv`
which is required by the dashboard.

### Step 3 — Run the dashboard
```bash
cd dashboard
python app.py
```
Open `http://127.0.0.1:8050` in your browser.

## Key Findings

- **Strong negative correlation** of -0.68 between load shedding hours and revenue
- **Stage 6** reduced average daily revenue by approximately 57% vs Stage 0
- **Total estimated revenue lost** over 3 months: R 89,000+
- **Each hour of load shedding** cost approximately R 186 in lost revenue
- **Recommendation:** A generator investment would likely break even within 3–4 months
